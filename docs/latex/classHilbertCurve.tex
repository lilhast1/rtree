\doxysubsection{Detailed Description}
Implements a d-\/dimensional Hilbert curve for mapping between points and Hilbert indices. 

This class provides functions to convert between d-\/dimensional points and their corresponding Hilbert curve index, as well as transposition operations used in Hilbert indexing.
