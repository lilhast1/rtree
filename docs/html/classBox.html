Detailed Description

Represents an axis-aligned hyper-rectangular box in N-dimensional space.

Provides functionality to check whether a point is inside the box and to iterate over all points on the perimeter.

Visit all points on the perimeter of the box.

Calls the provided function for each point lying on the box's perimeter. The perimeter includes any point where at least one coordinate is equal to the lower or higher bound of the box.
